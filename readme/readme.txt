 บทแนะนำระบบซอฟต์แวร์ชุมชนออนไลน์ Discuz! X
+----------------------------------+
Crossday Discuz! Board (เรียกสั้น ๆ ว่า Discuz! ซึ่งจดทะเบียนลิขสิทธิ์กับสำนักงานลิขสิทธิ์แห่งชาติจีนหมายเลข 2008SR11708) 
เป็นระบบซอฟต์แวร์สำหรับสร้างเว็บบอร์ดชุมชนออนไลน์ที่พัฒนาโดยบริษัท เทนเซ็นต์ คลาวด์ คอมพิวติ้ง (ปักกิ่ง) จำกัด ซึ่งเดิมคือบริษัท คอมเซนส์เทคโนโลยี (ปักกิ่ง) จำกัด (ตัวย่อภาษาอังกฤษ Comsenz) 
ในฐานะผู้ให้บริการซอฟต์แวร์และบริการชุมชนออนไลน์รายใหญ่ที่สุดในประเทศจีน ผลิตภัณฑ์ Discuz! ของ Comsenz 
เป็นผู้นำทั้งในและต่างประเทศในด้านฟังก์ชันการทำงาน เสถียรภาพ ความสามารถในการรองรับภาระงานและการรักษาความปลอดภัย จัดเป็นหนึ่งในระบบซอฟต์แวร์เว็บบอร์ดที่มีความสมบูรณ์และครอบคลุมการใช้งานมากที่สุดในโลก
ปัจจุบันได้รับการดูแลและพัฒนาต่อโดยบริษัท เน็ตเวิร์คเทคโนโลยีเหอเฟยเอ้อร์เต๋า จำกัด
