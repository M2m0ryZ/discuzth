เว็บบอร์ดดิสคัสไทย Facebook ดิสคัสไทย คำแนะนำในการติดตั้ง ดาวน์โหลด Discuz! X3.5 ภาษาไทย
การติดตั้งใหม่

  1. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดใน upload ไปยังเซิร์ฟเวอร์ของคุณ โดยใช้โปรแกรม FTP ตามสะดวกของคุณ เช่น โปรแกรม FileZilla เป็นต้น (ไม่จำเป็นต้องอัปโหลดโฟลเดอร์ readme และไฟล์ utility.html)
  2. เปิดเบราว์เซอร์เพื่อทำการติดตั้ง Discuz! X3.5 โดยไปที่ yourdomain.com/install เพื่อเริ่มต้นขั้นตอนการติดตั้ง แล้วทำตามหน้าจอที่ปรากฏ

ดูการติดตั้งแบบละเอียดได้ที่นี่: https://discuzthai.com/thread-40992-1-1.html


คําแนะนําในการอัปเกรด

อัปเกรดจาก X3.4

  1. ดาวน์โหลดไฟล์อัปเกรดได้ที่: https://discuzthai.com/thread-41059-1-1.html
  2. สำรองฐานข้อมูลและไฟล์โปรแกรม Discuz! และ UCenter เก่าของคุณ รวมไปถึงไฟล์ที่เกี่ยวข้องทั้งหมดในเซิร์ฟเวอร์ให้เรียบร้อยก่อนดำเนินการ
  3. ไปที่ “โฟลเดอร์รากเว็บบอร์ด/uc_server” แล้วสร้างโฟลเดอร์ “old” ขึ้นมาใหม่ (ตัวอย่างเส้นทางโฟลเดอร์: “โฟลเดอร์รากเว็บบอร์ด/uc_server/old”) แล้วทำการย้ายโฟลเดอร์ทั้งหมดที่อยู่ใน “โฟลเดอร์รากเว็บบอร์ด/uc_server/*.*” (ยกเว้นโฟลเดอร์ต่อไปนี้ห้ามย้าย: “data”) ไปที่ “โฟลเดอร์รากเว็บบอร์ด/uc_server/old” (หากมีไฟล์การกำหนดค่าเว็บไซต์ต่าง ๆ นอกเหนือจากไฟล์ของ UCenter รวมไปถึงไฟล์ .htaccess หรือ .user.ini ไม่ต้องย้ายไฟล์ดังกล่าว)
  4. อัปโหลดโปรแกรม UCenter 1.7.0 (อยู่ในไฟล์ ZIP ของ Discuz! X3.5 ตำแหน่งโฟลเดอร์ คือ upload/uc_server หากคุณใช้โปรแกรม FTP ในการอัปโหลด ใช้โหมดการอัปโหลดเป็นไบนารี) ไปที่ “โฟลเดอร์รากเว็บบอร์ด/uc_server” หากคุณได้รับแจ้งให้เขียนทับโฟลเดอร์ขณะอัปโหลด กรุณาเลือก "ใช่"
  5. อัปโหลดไฟล์ update_ucenter_adult.php ไปที่ “โฟลเดอร์รากเว็บบอร์ด/uc_server/install” และลบไฟล์ “โฟลเดอร์รากเว็บบอร์ด/uc_server/data/upgrade.lock” ออก
  6. เปิดเบราว์เซอร์ แล้วไปลิงก์นี้: “URLเว็บบอร์ด/uc_server/install/update_ucenter_adult.php” และปฏิบัติตามคำแนะนำตามหน้าจอที่ปรากฎเพื่อดำเนินการอัปเกรด เวลาในการอัปเกรดจะแตกต่างกันไปตามขนาดของข้อมูลและประสิทธิภาพของเซิร์ฟเวอร์
  7. กลับมาที่ โฟลเดอร์รากเว็บบอร์ด ของคุณ จากนั้น ให้ทำการย้ายโฟลเดอร์ทั้งหมด (ยกเว้นโฟลเดอร์ดังต่อไปนี้ห้ามย้าย: config, data และ uc_server) ไปที่โฟลเดอร์ old (ตัวอย่างเส้นทางโฟลเดอร์: “โฟลเดอร์รากเว็บบอร์ด/old”) หากมีไฟล์การกำหนดค่าเว็บไซต์ต่าง ๆ รวมถึงไฟล์ .htaccess หรือ .user.ini ไม่ต้องย้ายไฟล์ดังกล่าว
  8. อัปโหลดโปรแกรม Discuz! X3.5 (ไฟล์โปรแกรมทั้งหมดจะอยู่ในโฟลเดอร์ upload ของไฟล์ ZIP Discuz! X3.5 กรุณาใช้โหมดไบนารีหากคุณใช้โปรแกรม FTP ในการอัปโหลด) ไปยัง “โฟลเดอร์รากเว็บบอร์ด” บนเซิร์ฟเวอร์ของคุณ และโปรแกรม FTP จะแจ้งให้เขียนทับโฟลเดอร์ กรุณาเลือก "ใช่"
  9. ย้ายโฟลเดอร์ปลั๊กอินทั้งหมดที่อยู่ใน “โฟลเดอร์รากเว็บบอร์ด/old/source/plugin/*.*” กลับไปที่ “โฟลเดอร์รากเว็บบอร์ด/source/plugin” (ยกเว้นโฟลเดอร์ต่อไปนี้ห้ามย้าย: cloudcaptcha, cloudsearch, cloudstat, cloudunion, manyou, mobile, myapp, myrepeats, pcmgr_url_safeguard, qqconnect, qqgroup, security, soso_smilies, xf_storage, wechat) และย้ายโฟลเดอร์เทมเพลททั้งหมด (ยกเว้นโฟลเดอร์ต่อไปนี้ห้ามย้าย: “default”) ที่อยู่ใน “โฟลเดอร์รากเว็บบอร์ด/old/template/*.*” กลับไปที่ “โฟลเดอร์รากเว็บบอร์ด/template”
  10. อัปโหลดไฟล์ update_adult.php ไปที่ “โฟลเดอร์รากเว็บบอร์ด/install” และลบไฟล์ “โฟลเดอร์รากเว็บบอร์ด/data/update.lock” ออก
  11. เปิดเบราว์เซอร์ แล้วไปลิงก์นี้: “URLเว็บบอร์ด/install/update_adult.php” และปฏิบัติตามคำแนะนำตามหน้าจอที่ปรากฎเพื่อดำเนินการอัปเกรด เวลาในการอัปเกรดจะแตกต่างกันไปตามขนาดของข้อมูลและประสิทธิภาพของเซิร์ฟเวอร์

หากต้องการอัปเกรดจากเวอร์ชันอื่น กรุณาอัปเกรดเป็น X3.4 ก่อน: https://discuzthai.com/thread-40670-1-1.html
